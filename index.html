Częste pytania

Czy to naprawdę darmowe na zawsze?

Tak. Licencja MIT, kod na GitHubie. Nie ma wersji "Pro", nie planujemy też. Jeśli kiedyś chcielibyśmy płatnej wersji, fork zostaje publiczny i każdy może go używać.

Czy zawiera jakieś reklamy lub trackery?

Nie. Aplikacja nie wysyła żadnych danych. Asystent AI używa Twoich własnych kluczy API (opcjonalnie). Możesz zweryfikować w kodzie źródłowym.

Czy mogę zaufać czyszczeniu rejestru?

Każda zmiana w rejestrze jest poprzedzona automatycznym backupem (.reg). Możesz cofnąć każdą operację. Kod modułu registry_cleaner.py jest publiczny.

Dlaczego ufać niezależnemu projektowi a nie CCleanerowi?

Bo nasz kod możesz przeczytać. CCleanera nie. W 2017 r. CCleaner zawierał malware podpisane oficjalnym certyfikatem. Open source uniemożliwia takie sytuacje.

Jakie są wymagania systemowe?

Windows 10 (build 1809) lub 11, x64, ~50 MB miejsca na dysku, ~100 MB RAM podczas pracy. Działa bez uprawnień admina (z ograniczonymi funkcjami) lub z uprawnieniami (pełen zestaw).
