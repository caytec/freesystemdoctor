# FreeAndroidDoctor — product deck

Bilingual (PL + EN) 16-slide product presentation for FreeAndroidDoctor v2.0.0.

## Output

- `dist/FreeAndroidDoctor-deck-PL.pdf` — wersja polska
- `dist/FreeAndroidDoctor-deck-EN.pdf` — English version

Each PDF is 16 slides at 16:9 widescreen (254 mm × 142.875 mm = 1920×1080 px logical), ~1.2 MB.

## Source

| File | Purpose |
|---|---|
| `deck-pl.html` | Polish slide markup |
| `deck-en.html` | English slide markup |
| `style.css` | Shared visual styling (brand colors, layouts) |
| `build.py` | WeasyPrint render script |
| `assets/` | Icon, feature graphic, app screenshots (copies from `android/playstore/`) |

## Regenerate

```bash
pip install weasyprint
cd presentation
python3 build.py
```

Output lands in `presentation/dist/`.

## Updating screenshots

Screenshots come from `android/playstore/*.png`, generated by
`android/playstore/gen_assets.py`. After editing brand strings there:

```bash
cd android/playstore
python3 gen_assets.py
cp *.png ../../presentation/assets/
cd ../../presentation && python3 build.py
```
