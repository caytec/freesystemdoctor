## Full description (max 4000 chars)
```
FreeSystemDoctor is a free, open-source phone-cleaning and maintenance app for Android. No fake gigabytes, no "RAM Booster" placebo, no contact harvesting. Just real control over your storage, battery, and apps.

🔹 WHAT IT DOES

▸ Cleaner — clears own cache, leftover APK files, .tmp/.log files via MediaStore
▸ Recycle bin — browse and restore items moved to system trash (Android 11+)
▸ Hidden caches — removes the shadow folders that WhatsApp, Telegram, Spotify, Discord, Instagram, and TikTok keep around
▸ Duplicate files — SHA-256 comparison, identical copies grouped
▸ Large files — filter by type and size
▸ Storage by type — split into images, video, audio, documents, archives
▸ Folder analyzer — explore any folder via Storage Access Framework, find empty subfolders
▸ Similar photos — visually-close photographs detected automatically

🔹 APPS

▸ App manager — sort by size, uninstall in one tap
▸ Screen time — 7-day usage report per app
▸ Rarely used — apps you haven't opened in 30 days
▸ Permission audit — which apps have mic, camera, location
▸ APK backup — export an installed app to a .apk file
▸ Contacts + SMS backup — .vcf + .json to Downloads

🔹 SYSTEM

▸ Memory — real free RAM without force-killing (which hurts battery on Android)
▸ Battery — level, temperature, voltage, charging current (mA)
▸ Speed test — real download speed on a 10 MB sample
▸ Device info — SoC, ABI, sensors, security patch
▸ AI assistant — analyze device health (your API key, no data sent without your consent)

🔹 ADVANCED (Pro)

▸ App Vault — encrypted sandbox locked behind biometrics (AES/GCM, Android Keystore)
▸ Battery Alarms — notify at chosen charge thresholds (e.g. unplug at 80%, warn at 15%)
▸ App Insights — 7-day chart, recent installs, apps without a launcher icon
▸ Wi-Fi analyzer — SSID, channel, RSSI
▸ File shredder — secure overwrite before delete (DoD 5220.22-M)

🔹 WHY THIS APP

✓ All code public on GitHub (MIT License) — build the APK yourself and compare hashes
✓ Works without root, via official Android APIs — no warranty issues
✓ No "RAM Boost" (it hurts more than helps on Android 7.0+) — we show real values
✓ No data harvesting, no user accounts, no telemetry beyond the standard AdMob SDK
✓ Full PL and EN localization, not "Translate-quality"
✓ Modern stack: Kotlin + Jetpack Compose + Material 3, 60 FPS animations

🔹 PRO VERSION

A one-time tip unlocks 4 premium tools (Vault, Battery Alarms, App Insights, Shredder) and removes all ads. Every other feature stays free and unlimited forever.

🔹 ADS & PRIVACY

The free version contains AdMob ads (Google's standard ads SDK). On first launch we show a UMP consent dialog — you decide. The app itself doesn't log anything to any server. Full privacy policy: https://freesystemdoctor.pl/android/privacy.html

🔹 CONTACT

Bugs and suggestions: github.com/caytec/freesystemdoctor/issues
Email: coopaisolutions@gmail.com
```
