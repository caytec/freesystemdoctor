## Pełny opis (max 4000 znaków)
```
FreeAndroidDoctor to darmowa, open-source aplikacja do czyszczenia i konserwacji Twojego telefonu z Androidem. Bez fake-gigabajtów, bez „RAM boostera", bez wykradania kontaktów. Tylko realna kontrola nad pamięcią, baterią i aplikacjami.

🔹 CO POTRAFI

▸ Cleaner — czyści cache własne, leftover pliki APK, .tmp/.log z MediaStore
▸ Kosz — przeglądaj i przywracaj pliki usunięte przez system (Android 11+)
▸ Ukryte cache — kasuje shadow-foldery, które trzymają WhatsApp, Telegram, Spotify, Discord, Instagram i TikTok
▸ Duplikaty plików — porównanie SHA-256, grupowanie identycznych kopii
▸ Duże pliki — filtruj po typie i rozmiarze
▸ Storage by type — podział na obrazy, wideo, audio, dokumenty, archiwa
▸ Folder analyzer — przejrzyj dowolny folder przez Storage Access Framework, znajdź puste podfoldery
▸ Podobne zdjęcia — wykrywa wizualnie zbliżone fotografie

🔹 APLIKACJE

▸ App manager — sortuj wg rozmiaru, odinstaluj jednym tapnięciem
▸ Screen time — 7-dniowy raport czasu używania per aplikacja
▸ Rzadko używane — znajdź apki, których nie otworzyłeś od 30 dni
▸ Audyt uprawnień — które apki mają mikrofon, kamerę, lokalizację
▸ APK backup — wyeksportuj zainstalowaną aplikację do pliku .apk
▸ Backup kontaktów i SMS — eksport .vcf + .json do folderu Pobrane

🔹 SYSTEM

▸ Memory — realny free RAM bez wymuszania kill (które na Androidzie szkodzi baterii)
▸ Battery — poziom, temperatura, napięcie, prąd ładowania (mA)
▸ Speed test — prawdziwy test prędkości na próbce 10 MB
▸ Device info — SoC, ABI, czujniki, security patch
▸ Asystent AI — analiza kondycji urządzenia (Twój klucz API, dane nie wysyłane bez Twojej zgody)

🔹 ZAAWANSOWANE (Pro)

▸ App Vault — zaszyfrowany sejf chroniony biometrią (AES/GCM, Android Keystore)
▸ Battery Alarms — alarmy przy ustawionych progach naładowania (np. odłącz przy 80%, ostrzeż przy 15%)
▸ App Insights — wykres 7-dni, ostatnie instalacje, apki bez ikony w launcherze
▸ Wi-Fi analyzer — SSID, kanał, RSSI
▸ File shredder — bezpieczne nadpisanie pliku przed usunięciem (DoD 5220.22-M)

🔹 DLACZEGO TA APLIKACJA

✓ Cały kod publiczny na GitHubie (MIT License) — możesz zbudować APK sam i porównać hash
✓ Działa bez roota, oficjalnymi API Androida — bez utraty gwarancji
✓ Bez „RAM Boost" (na Androidzie 7.0+ szkodzi, nie pomaga) — pokazujemy realne wartości
✓ Bez sprzedaży danych, bez kont użytkowników, bez telemetrii poza standardowym SDK AdMob
✓ Pełna lokalizacja PL i EN, nie „przez Translate"
✓ Nowoczesny stack: Kotlin + Jetpack Compose + Material 3, animacje 60 FPS

🔹 WERSJA PRO

Jednorazowe wsparcie projektu odblokowuje 4 narzędzia premium (Vault, Battery Alarms, App Insights, Shredder) i usuwa wszystkie reklamy. Wszystkie inne funkcje pozostają darmowe i nielimitowane na zawsze.

🔹 REKLAMY I PRYWATNOŚĆ

Wersja darmowa zawiera reklamy AdMob (standardowy SDK Google'a). Przy pierwszym uruchomieniu pokazujemy okno zgody UMP — Ty decydujesz. Aplikacja sama nie loguje niczego do żadnego serwera. Pełna polityka prywatności: https://freeandroiddoctor.pl/privacy.html
```
